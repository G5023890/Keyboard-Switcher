# Third-Party Notices

This file documents third-party data, project-generated resources, and externally supplied assets included in Keyboard Switcher `v0.88`.

## Project-Provided Dictionary Data

Bundled files:

```text
KeyboardSwitcher/Resources/ru_auto_core_100k.tsv
KeyboardSwitcher/Resources/ru_manual_extended_300k.tsv
KeyboardSwitcher/Resources/en_auto_core_50k.tsv
KeyboardSwitcher/Resources/en_manual_extended_200k.tsv
KeyboardSwitcher/Resources/short_words_auto_whitelist.tsv
```

Source:

These Russian and English scoring dictionaries are project-provided bundled resources maintained for Keyboard Switcher. The TSV dictionaries are split into automatic core dictionaries and manual delta dictionaries.

Modification note:

The dictionaries are compact local scoring resources for layout-correction candidates. Automatic correction uses only the auto core dictionaries and short-word whitelist; manual delta dictionaries are used only for manual correction/candidate selection and intentionally exclude words already covered by automatic layers. They are intentionally small and are not a large third-party NLP runtime.

## Project-Provided Technical Term Data

Bundled files:

```text
KeyboardSwitcher/Resources/technical_never_correct.tsv
```

Source:

These technical terms and technical-token rules were supplied for Keyboard Switcher to reduce false corrections around macOS, iOS, SwiftUI, Core ML, Xcode, APIs, bundle identifiers, filenames, paths, versions, and command-like text.

Status:

Project-provided data.

## Project-Generated Local ML Model

Bundled file:

```text
KeyboardSwitcher/Resources/CorrectionSafetyClassifier.mlmodel
```

Source:

This Core ML model is generated by the Keyboard Switcher project from synthetic/project training data. It is used locally in shadow diagnostics mode in `v0.88` and does not replace the deterministic correction rules.

Status:

Project-generated model.

## Evaluated But Not Bundled

The following resources were considered during development or used as references, but are not bundled as current dictionaries in `v0.88`:

- `wordfreq` 3.1.1 by Robyn Speer: https://github.com/rspeer/wordfreq
- `pymorphy3`: https://github.com/no-plagiarism/pymorphy3
- `first20hours/google-10000-english`: not included because its license notice cautions against commercial use without licensing from the Linguistic Data Consortium.
- `dwyl/english-words`: not included because the app does not currently need a very large English word list.
- `hingston/russian` and the University of Leeds Russian Internet Corpus frequency list: not included after replacing the Russian frequency resource.
- `MichaelWehar/Public-Domain-Word-Lists`: not included after replacing the English common-word resource.
